# Hetzner Scraper

Scrapes the Hetzner Servers and sends a Discord Webhook if the price has changed.

## Usage

### Setup

```bash

git clone https://github.com/jodu555/Hetzner-Scraper.git

cd Hetzner-Scraper

cp .env.example .env

# Edit .env

npm install

npm install ts-node

```

### Run

```bash
npm run start
```
